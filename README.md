## ðŸ“– Project Description

> **Engremaq** is an e-commerce website focused on selling parts and accessories for tractors. It offers a complete online shopping experience where customers can view, search for, and purchase products related to the world of tractors. Additionally, the store provides a contact form for users to get in touch easily.

This repository contains the source code for all pages of the site, including features like a shopping cart, product search, and an intuitive navigation interface.

## âœ¨ Features

- **Homepage**: A welcome page with links to the main sections of the site.
- **Product Search**: A search tool to find specific products in the catalog.
- **Product Catalog**: Products divided into categories such as "Accessories," "Pumps," etc.
- **Shopping Cart**: Products can be added to the cart to facilitate the purchase.
- **Contact Form**: Customers can send messages directly to the store to ask questions or request more information.
- **Social Media Links**: Connections to Instagram and Facebook for customer interaction.
